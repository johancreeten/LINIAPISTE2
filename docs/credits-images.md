# Crédits images

Date de préparation : 2026-07-05.

## Note importante
Pour débloquer la demande d’extraction GitHub, les visuels du prototype sont désormais des fichiers SVG locaux textuels. GitHub peut donc afficher leur diff sans bloquer sur des fichiers binaires.

Ces SVG sont des visuels temporaires inspirés de la direction artistique LINIA (papier chaud, kraft, lavande, sauge, fleurs séchées, trait de parcours). Ils remplacent les WebP binaires dans cette branche de revue. Avant mise en production, ils pourront être remplacés par de vraies photos Pexels exportées en WebP ou JPEG.
